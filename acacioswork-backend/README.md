# AcaciosWork - Backend (El Cerebro)

## Descripción
Este es el núcleo del sistema AcaciosWork, desarrollado con Spring Boot. Proporciona una API REST para la gestión de inventario, ventas y usuarios.

## Tecnologías
- Java 25+
- Spring Boot 4.x
- Spring Data JPA
- Spring Security + JWT
- MySQL

## Estructura de Paquetes
- `controller`: Endpoints de la API.
- `service`: Lógica de negocio.
- `repository`: Acceso a datos.
- `model`: Entidades de la base de datos.
- `dto`: Objetos de transferencia de datos.
- `config`: Configuraciones de seguridad y CORS.
