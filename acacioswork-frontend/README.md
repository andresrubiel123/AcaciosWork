# AcaciosWork - Frontend Web

## Descripción
Interfaz de usuario web para el sistema AcaciosWork. Construida con HTML, CSS y JavaScript (Vanilla).

## Características
- Diseño moderno y responsivo.
- Consumo de API REST mediante `fetch`.
- Gestión de estado local y autenticación por JWT.

## Estructura
- `index.html`: Punto de entrada.
- `login.html`: Inicio de sesión.
- `dashboard.html`: Panel de control.
- `css/`: Estilos globales.
- `js/`: Lógica de la aplicación y conexión API.

## Ejecución
Simplemente abre `index.html` en un navegador o usa un servidor estático como Live Server.
