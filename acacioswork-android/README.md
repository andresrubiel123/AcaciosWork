# AcaciosWork - Android App

## Descripción
Aplicación móvil nativa para el sistema AcaciosWork, desarrollada en Kotlin.

## Tecnologías
- Kotlin
- Android SDK
- Retrofit (API Client)
- ViewModel & LiveData (MVVM)
- Jetpack Navigation

## Estructura
- `ui/`: Activities y Fragments.
- `network/`: Cliente de API y servicios.
- `viewmodel/`: Lógica de la interfaz de usuario.
- `model/`: Clases de datos.
- `repository/`: Repositorios de datos.

## Ejecución
Requiere Android Studio para compilar y ejecutar en un emulador o dispositivo físico.
